## API

### props

| props       | 字段                                    |
| ----------- | --------------------------------------- |
| percent     | 当前进度                                |
| showInfo    | 是否展示当前进度信息                    |
| status      | 进度条的状态 success，exception，normal |
| strokeColor | 进度条的颜色                            |
| trailColor  | 进度条轨道的颜色                        |
| strokeWidth | 设置进度条的宽度                        |
| width       | 设置 circle 进度条画布的宽度            |
